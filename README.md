## Installation
Add this line to your application's Gemfile:

```ruby
gem 'spree_dyn_related_products', github: 'MateoLa/spree_dyn_related_products'

```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install spree_dyn_related_products
```
